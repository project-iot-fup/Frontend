### Instalación 🔧

#### Importante Tener instalado Python y NodeJs 📣

#### **Frontend** 📦

```bash
# Clonar el repositorio
$ git clone https://github.com/project-iot-fup/Frontend
$ cd Frontend
# Instalar las dependencias
$ npm install
# Correr proyecto
$ npm start
```

### **RaspberryPi 3**

```bash
# Conectar a la RaspberryPi 3
$ ssh pi@raspberrypi.local
# Usuarion: pi
# Contraseña: pi2022*
```
